# Stage 01 — Outline (Layer 2 contract)

## Inputs
- Layer 4 (working): ../../setup/brief.md          # the one-line topic brief for this run
- Layer 3 (reference): references/structure.md       # the fixed outline shape

## Process
1. Read the topic brief.
2. Produce a structured outline following structure.md: a working title, a one-line
   angle, 3–4 section headings, and one supporting point under each.
3. Keep it to the outline only — no prose. This is the cheap place to course-correct.
