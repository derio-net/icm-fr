# Setup Questionnaire

Answered once, at workspace setup. The answers populate `_config/` (Layer 3) so
every later run inherits them. Configure the factory, not the product.

1. **Domain** — what does this workspace produce? (e.g. weekly competitor digest)
2. **Voice / style** — tone, register, audience. → `_config/voice.md`
3. **Structure** — the fixed shape of the deliverable. → `_config/structure.md`
4. **Constraints** — hard limits (length, format, must-include/avoid).
5. **Review points** — at which stage boundaries does a human always check?
